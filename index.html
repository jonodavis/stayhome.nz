COVID-19 Event Risk Calculator

The calculator below is used to estimate the risk involved with attending a public event during the COVID-19 outbreak. Change the input values manually or use the common values box to see how the risk changes depending on population, number of circulating cases, and event size.

Up to date information on COVID-19 is available from the WHO website. A well-sourced map of confirmed and active COVID-19 cases can be found here.
